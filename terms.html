利用規約

本利用規約（以下「本規約」といいます。）は、株式会社バリーが提供するスマートフォン向けゲームアプリ「すもうとり」および関連サービス（以下「本サービス」といいます。）の利用条件を定めるものです。ユーザーは本規約に同意したうえで本サービスを利用するものとします。

第1条（定義）

本規約において使用する用語の定義は、以下のとおりとします。

当社
本サービスを提供する株式会社バリー（所在地：東京都千代田区神田須田町1-32-7）をいいます。
本サービス
当社が提供するスマートフォン向けゲームアプリ「すもうとり」および関連サービスをいいます。
ユーザー
本規約に同意したうえで本サービスを利用する個人をいいます。
利用契約
本規約に基づき当社とユーザーとの間で成立する、本サービスの利用に関する契約をいいます。
歩数データ
ユーザーの端末のHealthKit等から取得する歩数情報をいいます。
くじポイント
本サービス内で取得できるゲーム内ポイントをいい、抽選等に利用できます。
金のチケット
本サービス内で取得できるゲーム内チケットをいい、ギフト交換等に利用できます。
オファーウォール
本サービス内において第三者が提供する広告サービスをいいます。
広告事業者
広告配信またはオファーウォールを提供する第三者をいいます。
賞品
ユーザーが抽選、交換等により取得できる電子マネー、ギフトコード、商品等をいいます。

第2条（サービス内容）

  1. 本サービスは、iPhoneの歩数データと連動してキャラクター（すもうとり）を育成するスマートフォン向けゲームアプリです。
  2. ユーザーは、卵から育つすもうとりをお世話し、歩数や連勝に応じて番付を上げ、大関、横綱を目指すことができます。
  3. ユーザーは、本サービス内で取得したくじポイントまたは金のチケットを、当社が定める方法で抽選や賞品交換に利用できます。
  4. 本サービスは無料で利用でき、アプリ内課金はありません。広告収益により運営されています。
  5. 本サービスの内容、仕様、提供方法は当社の判断により変更される場合があります。

第3条（規約の適用）

  1. 本規約は、本サービスの利用に関して当社とユーザーとの間の一切の関係に適用されます。
  2. 当社が別途定めるガイドラインや注意事項等は本規約の一部を構成するものとします。
  3. 本サービスにおける個人情報の取扱いについては、プライバシーポリシーをご確認ください。

第4条（歩数データの取扱い）

  1. 本サービスは、端末のHealthKit等から歩数データを取得し、すもうとりの育成、番付、日々のきろく等のゲーム進行に利用します。
  2. ユーザーは、初回起動時等に歩数データの取得許可を求められます。許可は端末設定からいつでも変更できます。
  3. 歩数データの取得を許可しない場合、本サービスの一部機能をご利用いただけない場合があります。
  4. 取得した歩数データは、本サービスの提供に必要な範囲で利用します。詳細はプライバシーポリシーをご確認ください。

第5条（くじポイント・金のチケット）

  1. ユーザーは、本サービスを利用することでくじポイントおよび金のチケットを取得できます。
  2. くじポイントおよび金のチケットは、すもうとりが土俵に産む星の回収、広告の利用、その他当社が指定する方法により付与される場合があります。
  3. 白星はくじポイント、金星は金のチケット、大金星はくじポイントと金のチケットの両方の取得対象となります。黒星はペナルティとして扱われます。
  4. 付与されるくじポイントおよび金のチケットの数量、条件、頻度は、当社の判断により変更される場合があります。
  5. 金のチケットの有効期限は、最終取得から60日間です。有効期限を過ぎた未使用分は失効します。
  6. 当社は、くじポイントについても有効期限その他の利用条件を設定できるものとします。
  7. くじポイントおよび金のチケットは現金化することはできません。
  8. 当社はくじポイントおよび金のチケットの再発行、返還を行いません。

第6条（賞品）

  1. ユーザーは、当社の定める条件に従い、金のチケットをえらべるPayを通じたAmazon、Apple、コンビニ等のギフトに交換できます。
  2. 賞品内容、交換条件、必要な金のチケット数は当社または提携先が定めるものとし、時期により変更される場合があります。
  3. 抽選またはギフト交換の利用には、電話番号によるSMS認証が必要となる場合があります。
  4. 当社は賞品の品質、有効性について保証しません。
  5. 賞品の再発行、交換、返金は行いません。
  6. 未成年者は法定代理人の同意が必要です。

第7条（SMS認証）

  1. ユーザーは、抽選、ギフト交換、「お得」タブの利用時に、電話番号によるSMS認証を求められる場合があります。
  2. 当社は、認証に利用した電話番号について、SHA-256ハッシュおよび末尾4桁のみを保存します。
  3. 電話番号は、報酬の不正取得防止、複数アカウント利用の抑止、本人確認のために利用します。
  4. SMS認証を完了しない場合、本サービスの一部機能をご利用いただけない場合があります。

第8条（アカウント）

  1. ユーザーは自己の責任でアカウントを管理するものとします。
  2. ユーザーは、メールアドレスまたは電話番号を利用して、機種変更時のデータ引き継ぎを行える場合があります。
  3. 本サービスのデータは、端末およびサーバ（Firebase）にバックアップされる場合があります。
  4. アカウントの貸与、譲渡、売買は禁止します。
  5. アカウントの不正利用に関して当社は責任を負いません。

第9条（複数アカウントの禁止）

同一ユーザーによる複数アカウントの作成または利用は禁止します。当社は不正と判断した場合、以下の措置を行うことがあります。

  • くじポイント・金のチケットの没収
  • 賞品交換停止
  • アカウント停止
  • 利用契約解除

第10条（不正な歩数取得の禁止）

ユーザーは以下の行為を行ってはなりません。

  • 歩数偽装アプリ・デバイスの利用
  • 振動装置等の機械的な歩数水増し行為
  • ヘルスケアデータの手動編集による不正水増し
  • その他不正な手段による歩数の取得

第11条（自動操作・BOTの禁止）

ユーザーは以下の行為を行ってはなりません。

  • BOT
  • 自動クリックツール
  • マクロ
  • 不正プログラム

これらを利用してくじポイント、金のチケット、賞品その他の利益を取得する行為は禁止します。

第12条（VPN・Proxyの禁止）

ユーザーは以下の行為を行ってはなりません。

  • VPN
  • Proxy
  • IP偽装
  • 位置情報偽装

これらを利用してポイント取得条件や広告成果条件を不正に満たす行為は禁止します。

第13条（エミュレーターの禁止）

当社が認めないエミュレーター環境での利用は禁止します。

第14条（端末識別および不正検知）

当社は不正利用防止のため以下の情報を利用することがあります。

  • Firebase Auth UID
  • 独自deviceId
  • 電話番号のSHA-256ハッシュおよび末尾4桁
  • IPアドレス
  • 利用ログ
  • 広告識別子
  • 歩数データの統計値

これらの情報により不正利用を検知する場合があります。

第15条（オファーウォールおよび広告）

  1. 本サービスでは、Google Mobile Ads、Unity Ads、FIVE、InMobi等の広告配信サービスを利用する場合があります。
  2. 本サービス内の「お得」タブでは、ユーザーが選択した場合のみSkyFlagのオファーウォールが表示されます。
  3. オファーウォールの広告利用により報酬が付与される場合があります。
  4. 報酬付与の可否は広告事業者の判定に従うものとします。
  5. 広告はスキップ可能です。ただし、広告の仕様は広告事業者または配信状況により変更される場合があります。

第16条（ポイント・チケット取消）

以下の場合、当社はくじポイントおよび金のチケットを取消または失効させることができます。

  1. 不正利用が確認された場合
  2. 利用規約違反
  3. 広告事業者による成果否認
  4. システムエラーによる誤付与
  5. 金のチケットの有効期限が経過した場合
  6. その他当社が必要と判断した場合

第17条（サービス変更）

当社はユーザーに通知することなく本サービスの内容を変更できます。

第18条（サービスの中断）

  • 通信障害
  • 自然災害
  • サイバー攻撃
  • その他当社が必要と判断した場合

第19条（禁止事項）

  1. 法令違反
  2. 公序良俗違反
  3. 不正アクセス

  7. 不正広告行為
  8. 報酬、賞品、アカウント等の譲渡、売買、換金

第20条（利用停止）

当社はユーザーが本規約に違反した場合以下を行うことができます。
