プライバシーポリシー

株式会社バリー（以下「当社」といいます。）は、当社が提供するスマートフォン向けゲームアプリ「すもうとり」および関連サービス（以下「本サービス」といいます。）において、ユーザーの個人情報および歩数データの重要性を認識し、その保護を徹底するため、本プライバシーポリシーを定めます。

第1条（基本方針）

当社は、個人情報の保護に関する法令および関連ガイドラインを遵守し、ユーザーの個人情報を適切に取り扱います。

また、個人情報の漏えい、滅失、毀損等を防止するため、合理的な安全対策を講じます。

第2条（取得する情報）

当社は、本サービスの提供にあたり、以下の情報を取得する場合があります。

1. 健康・フィットネス情報

  • 歩数（HealthKit）

歩数は、すもうとりの育成、番付、日々のきろく等、本サービスの中核機能を提供するために利用します。

2. 識別子

  • Firebase Auth UID
  • 独自deviceId（Keychain）
  • idfv（iOS）
  • Android ID

3. 連絡先情報

  • 電話番号（SHA-256ハッシュおよび末尾4桁のみ）
  • メールアドレス（アカウント登録またはお問い合わせ時）

電話番号は、抽選、ギフト交換、「お得」タブの利用時に行うSMS認証および報酬の不正取得防止のみに使用します。

4. 端末情報

  • OS
  • 端末機種名
  • IPアドレス
  • 言語設定

5. アプリ利用情報

  • アプリの利用履歴
  • ゲームプレイ履歴
  • くじポイント・金のチケット取得履歴
  • 起動回数
  • 金星収集マイルストーン
  • 広告閲覧履歴
  • クラッシュログ

6. ギフト交換・お問い合わせ情報（必要な場合）

  • ギフト交換履歴
  • お問い合わせ内容
  • 本人確認に必要な情報

第3条（利用目的）

当社は取得した情報を以下の目的で利用します。

  1. 本サービスの提供および運営
  2. 歩数データに基づくすもうとりの育成、番付、日々のきろく等の機能提供
  3. Firebase Authによるアカウント管理および機種変更時のデータ引き継ぎ
  4. くじポイント・金のチケットの管理およびギフト交換の提供
  5. SMS認証による報酬の不正取得防止
  6. 1端末1ユーザー制限、機種変更なりすまし防止等の不正対策
  7. ユーザーサポート対応
  8. サービス改善および品質向上
  9. 広告配信および広告効果測定
  10. UA計測および広告キャンペーン効果測定

第4条（ヘルスケアデータの取扱い）

  1. 本サービスは、HealthKitから歩数データを取得します。
  2. 歩数データの取得には、ユーザーの明示的な許可が必要です。許可は端末設定から随時変更できます。
  3. 取得した歩数データは、本サービスの育成、番付、日々のきろく等の機能に利用します。
  4. 歩数データを広告配信のために利用することはありません。
  5. 歩数データはユーザーの同意なく広告事業者へ提供しません。

第5条（外部サービスの利用）

当社は、本サービスにおいて以下の外部サービスを利用する場合があります。これらのサービス提供事業者は、ユーザーの端末情報や広告識別子等を取得する場合があります。

1. 認証・データ管理

Firebase Authentication / Firestore（Google LLC）

利用目的

  • ユーザー認証
  • アカウント情報、ゲーム進行、残高情報の保存
  • 端末およびサーバでのバックアップ
  • メールアドレスまたは電話番号による機種変更時のデータ引き継ぎ

2. 広告効果測定・UA計測

Adjust（Adjust GmbH）

利用目的

  • 広告効果測定
  • 広告成果の分析
  • idfvまたはAndroid IDを用いたS2S計測
  • 起動回数および金星収集マイルストーンの計測

3. 広告配信

以下の広告配信サービスを利用する場合があります。

  • Google Mobile Ads（Google LLC）
  • Unity Ads（Unity Technologies）
  • FIVE（LINEヤフー株式会社）
  • InMobi（InMobi Pte Ltd.）

利用目的

  • 広告の表示
  • 広告効果の測定
  • ユーザーに適した広告の配信

4. オファーウォール

SkyFlag

利用目的

  • 「お得」タブにおけるオファーウォールの表示
  • ユーザーが選択した広告案件の成果判定
  • 報酬付与および不正対策

SkyFlagは、ユーザーが「お得」タブを選択した場合のみ表示されます。

5. ギフト交換

えらべるPay

利用目的

  • 金のチケットのギフトへの交換
  • 交換履歴の確認

各サービスのプライバシーポリシーについては、各事業者のサイトをご確認ください。

第6条（第三者提供）

当社は、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命・身体・財産の保護のため必要な場合
  4. 個人を特定できない形の統計情報として提供する場合
  5. 本サービスの広告配信、UA計測、オファーウォール、ギフト交換に必要な範囲で外部サービス事業者へ提供する場合

第三者共有の対象となる主な事業者は、Google Mobile Ads、Unity Ads、FIVE、InMobi、SkyFlag、Adjustです。

第7条（Cookie等の利用）

本サービスでは、利用状況の分析および広告配信のため、Cookieまたはこれに類する技術を利用する場合があります。

第8条（情報の管理）

当社は、個人情報の漏えい、滅失または毀損を防止するため、適切な安全管理措置を講じます。

本サービスのデータは端末およびFirebaseサーバに保存・バックアップされ、メールアドレスまたは電話番号による本人確認を通じて機種変更時に引き継げる場合があります。

第9条（外部サイト）

本サービスからリンクされる外部サイトにおける個人情報の取扱いについて、当社は責任を負いません。

第10条（個人情報の開示・訂正等）

ユーザーは、当社に対して以下の請求を行うことができます。

  • 個人情報の開示
  • 訂正
  • 削除
  • 利用停止

当社は本人確認のうえ、合理的な期間内に対応します。

第11条（未成年者の利用）

未成年者が本サービスを利用する場合は、保護者の同意を得たうえで利用するものとします。

第12条（ポリシーの変更）

当社は、法令の変更またはサービス内容の変更に応じて、本ポリシーを変更する場合があります。変更後の内容は、本サービスまたは当社ウェブサイトに掲載した時点で効力を生じます。
